Project Overview

Project Name

Jacked Platypus Events Hub

Function and Purpose

Jacked Platypus Events Hub is a web application that promotes a local DJ and event brand. The site will showcase upcoming events, past shows, media highlights, and booking information. The goal is to build a professional online presence, help users discover events, and allow promoters to request bookings easily.

Intended Users

  • Students and young adults at UNC Charlotte looking for events.
  • Event organizers and promoters needing booking info.
  • Existing followers wanting updates, photos, and media.

Overview of Website Content

  • Home page with intro and featured event
  • About page with story and background
  • Events page listing shows
  • Media page with photos and clips
  • Contact / Booking page with a form

Page Design Details

1. Home Page (index.html)

  • Purpose: Introduce the brand and highlight an upcoming event.
  • Audience: All visitors.
  • Content: Hero, intro, featured event.
  • Data Entry: None.
  • Links/Buttons: “View Events”, “Book Now”.
  • Actions: Navigation to Events or Contact.

2. About Page

  • Purpose: Tell the DJ’s story.
  • Content: Bio, background, fun facts.
  • Interactivity: Accordion UI section.

3. Events Page

  • Purpose: Show event listings.
  • Content: Event cards with date and location.
  • Interactivity: Filtering and detail expanding.
  • AJAX: Load events from JSON.

4. Media / Gallery Page

  • Purpose: Display photos and clips.
  • Interactivity: Lightbox/zoom gallery.

5. Contact / Booking Page

  • Purpose: Booking form.
  • Fields: Name, Email, Event Type, Date, Message.
  • Validation: Required fields + email format.
